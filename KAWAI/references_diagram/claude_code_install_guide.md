# ① ターミナル編

# ワークフロー
- 以下の掲載内容を図解してください

# 絶対ルール
- すべてのテキストは日本語で表記する（英語禁止）
- コマンド部分のみ英語OK
- 配色は指定の3色のみ
- モダンでミニマルなデザイン。余白を大きく活かす
- 文字は大きく、読みやすく。小さい文字の詰め込み禁止
- フラットデザイン、影なし
- 各カード内の情報はアイコン1つ＋ラベル最大5文字に厳守

# ルール
- 画質: 4K高画質
- 1枚の画像にまとめる
- アスペクト比: 16:9（横長）
- 文字量は極限まで少なく。カード内ラベルは3〜5文字
- アイコンで直感的に伝える。テキストに頼らない

# 配色
- 背景: #E8784D（Claudeオレンジ）
- カード: #FFFFFF（角丸24px）
- テキスト: #1A1A1A
- アクセント: #D4612C（濃いオレンジ、ステップ番号用）
- ボトムバー: #D4612C背景、白文字

# レイアウト
- 上部にタイトル「ターミナルからインストール」白文字、太字
- 中央に横並びの3枚の白い角丸カード（カード間に余白たっぷり）
- 各カードの上部にステップ番号の丸アイコン（濃いオレンジ）
- カード間を右向き矢印でつなぐ
- 各カード内はアイコン（大きく）＋ラベル（3〜5文字）のみ
- 画像下部にボトムバー: コマンド全文を等幅フォントで1行表示

# スタイル
- フラットベクターイラスト
- 各カード内のアイコンはカード面積の50%以上を占める大きさ
- ラベルはアイコン直下に1行、太字
- ステップ番号は丸い円の中に白文字の数字

=== 掲載する内容 ===

タイトル: ターミナルからインストール

ステップ1:
アイコン: ターミナルウィンドウ（黒い画面に＞カーソル）
ラベル: 「開く」

ステップ2:
アイコン: 下向き矢印のインストールアイコン
ラベル: 「実行」

ステップ3:
アイコン: Claudeロゴマーク風の光るアイコン
ラベル: 「起動」

ボトムバー: curl -fsSL https://claude.ai/install.sh | bash
