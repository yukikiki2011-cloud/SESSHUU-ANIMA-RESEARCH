# ワークフロー
- 「掲載する内容」をじっくり読んでください。
- 中心テーマと主要カテゴリを抽出してください。
- 放射状のマインドマップとして図解してください。

# 絶対ルール
- すべてのテキストは日本語で表記する（英語禁止）
- 配色は最大3色まで。鮮やかで目を引く色を選ぶ
- モダンでミニマルなデザイン。余白を活かす
- 情報の階層を明確にする（タイトル→サブタイトル→本文）
- 文字は大きく、読みやすく。小さい文字の詰め込み禁止
- フラットデザイン、影なし、グラデーション最小限

# ルール
- 画質: 4K高画質
- 資料に含まれていない文字は掲載しない
- 自然な改行位置にする
- 1枚の画像にまとめる
- すべて日本語
- 文字量は少ない
- アスペクト比: 1:1（正方形）

# レイアウト
- 中央に大きな円でメインテーマを配置
- メインテーマから放射状に第1階層のブランチ（3〜6本）を伸ばす
- 各ブランチの先にサブトピックを円形ノードで配置
- 第2階層はさらに細い線で枝分かれ（必要に応じて）
- ブランチは有機的な曲線で描く
- ブランチは3色を交互に割り当てる

# 配色
- メインカラー: #2196F3（ブルー）
- サブカラー: #FF5722（オレンジレッド）
- アクセント: #4CAF50（グリーン）
- 中心テーマ: #1A1A2E（ダーク背景 + 白文字）
- 背景: #FFFFFF

# スタイル
- 有機的で柔らかい印象
- ブランチ線は太さ4px→2pxへ先細り
- ノードは角丸長方形
- アイコンやイラストは使わず、テキストのみ
