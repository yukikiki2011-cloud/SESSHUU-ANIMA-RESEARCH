# ワークフロー
- 「掲載する内容」をじっくり読んでください。
- 要素間の関係性（依存、影響、連携）を抽出してください。
- ネットワーク図（ノード＋エッジ）として図解してください。

# 絶対ルール
- すべてのテキストは日本語で表記する（英語禁止）
- 配色は最大3色まで。鮮やかで目を引く色を選ぶ
- モダンでミニマルなデザイン。余白を活かす
- 情報の階層を明確にする（タイトル→サブタイトル→本文）
- 文字は大きく、読みやすく。小さい文字の詰め込み禁止
- フラットデザイン、影なし、グラデーション最小限

# ルール
- 画質: 4K高画質
- 資料に含まれていない文字は掲載しない
- 自然な改行位置にする
- 1枚の画像にまとめる
- すべて日本語
- 文字量は少ない
- アスペクト比: 1:1（正方形）

# レイアウト
- 中心に最重要ノード（大きめの円）
- 周囲に関連ノードを配置
- ノード間を線（エッジ）で接続
- 関係の強さに応じて線の太さを変える（太い＝強い関係）
- 双方向の関係は両端矢印、一方向は片端矢印
- ノードの大きさは重要度に比例
- グループがある場合は薄い色の囲みでグルーピング

# 配色
- メインカラー: #2196F3（ブルー）
- サブカラー: #FF5722（オレンジレッド）
- アクセント: #4CAF50（グリーン）
- ノード文字: #FFFFFF（濃色背景上）
- 背景: #FFFFFF
- エッジ線: #333333

# スタイル
- クリーンで整理された印象
- ノードは正円、影なし
- 線の交差を最小限にするレイアウト
- 余白を十分に取る
